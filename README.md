# ইক্বরা (Iqra)

> **"পাঠ করুন আপনার পালনকর্তার নামে যিনি সৃষ্টি করেছেন।"**
> — সূরা আলাক: ১

## বই পরিচিতি

**ইক্বরা** একটি অনুপ্রেরণামূলক ইসলামিক বই যা বাংলাভাষী মুসলিম তরুণদের জন্য লেখা।

এই বইটি বলে—ইসলামের প্রথম আদেশ ছিল 'পড়ো'। শুধু ধর্মীয় কিতাব নয়, বরং পুরো মহাবিশ্বকে পড়ার আদেশ। আজকের মুসলিম তরুণ যদি সেই আদেশ পালন করে, তাহলে সে একদিন দ্বীনেও সেরা, দুনিয়াতেও নেতা।

## বিষয়বস্তু

- ইকরার প্রকৃত অর্থ
- মুসলিম বিজ্ঞানীদের গর্বিত উত্তরাধিকার
- সাহাবীদের উদ্যোক্তামূলক মানসিকতা
- প্রযুক্তি ও ইসলাম
- হালাল ব্যবসা ও স্বনির্ভরতা
- ডিজিটাল জিহাদ
- ইকরা জেনারেশনের শপথ
