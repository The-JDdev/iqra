# ইক্বরা (Iqra)

> **"পাঠ করুন আপনার পালনকর্তার নামে যিনি সৃষ্টি করেছেন।"**  
> — সূরা আলাক: ১

## পরিচয়

**ইক্বরা** একটি অনুপ্রেরণামূলক ইসলামিক বই যা বাংলাভাষী মুসলিম তরুণদের জন্য।

বইটি বলে — ইসলামের প্রথম আদেশ ছিল 'পড়ো'। শুধু ধর্মীয় কিতাব নয়, পুরো মহাবিশ্বকে পড়ার আদেশ।

১. ইকরা — প্রথম আদেশের গভীরতা

৩. মুসলিম বিজ্ঞানীদের উত্তরাধিকার  
৪. সাহাবীরা ছিলেন উদ্যোক্তা

৬. প্রযুক্তি — ইসলামের হাতিয়ার  
৭. কুরআন — মহাবিশ্বের ব্যবহারবিধি  
৮. মাদ্রাসা — লঞ্চিং প্যাড  
৯. হালাল ব্যবসা ও স্বনির্ভরতা  
১০. নেতৃত্ব  
১১. ডিজিটাল জিহাদ  
১২. ইসলামের সেবা  
১৩. হালাল অর্থনীতি  
১৪. ইকরা জেনারেশন — শপথ
